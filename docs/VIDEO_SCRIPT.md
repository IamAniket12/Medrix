## Visual Style Guidelines

**Color Palette**:
- Primary: Medical blue (#0066CC)
- Accent: Health green (#00CC66)
- Background: Clean white/light gray
- Text: Dark gray (#333333)

**Typography**:
- Headlines: Bold, modern sans-serif (Inter/Poppins)
- Body: Clean, readable (system fonts)
- Code: Monospace when showing technical details

**Animation Style**:
- Smooth transitions (300-500ms)
- Slide-in effects for stats
- Gentle hover states for interactive elements
- Loading animations that show AI "thinking"

**Demo Recording Tips**:
1. Use screen recording software (OBS, Loom, or QuickTime)
2. Record at 1080p minimum resolution
3. Clean up desktop/browser tabs before recording
4. Use cursor highlighting for better visibility
5. Rehearse 3-4 times before final recording
6. Add subtle background music (royalty-free, low volume)
7. Use professional microphone for narration
8. Add subtle zoom-ins on key UI elements
9. Keep camera segments short and energetic
10. Export at 60fps for smooth playback
